<h1> Aircraft </h1>

<h3> Цель игры </h3>

- Пройти сквозь как можно большее количество сфер
- Посадить самолет, не разбив его

<h3> Управление </h3>


  <li>W/S - для тангажа</li>
  <li>Q/E - для рыскания</li>
  <li>A/D - для крена</li>
  <li>U - газ</li>
  <li>Space - тормоз на земле</li>
  <li>J - отключение двигателя (удержание)</li>
</ul>
